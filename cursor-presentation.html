Cursor'ı İndir

🤖 Cursor Nedir?

Cursor, AI teknolojisiyle güçlendirilmiş yeni nesil kod editörüdür. VS Code'un tüm özelliklerini barındırırken, yapay zeka destekli kod yazma, düzenleme ve hata ayıklama yetenekleri sunar.

🧠

AI Kod Asistanı

GPT-4 entegrasyonu ile akıllı kod önerileri ve otomatik kod tamamlama

⚡

Hızlı Geliştirme

Kodunuzu anlayarak context-aware öneriler ve düzeltmeler

🔧

VS Code Uyumlu

Tüm VS Code eklentileri ve ayarlarınızla tam uyumluluk

✨ Cursor'ın Temel Özellikleri

1

🎯 Akıllı Kod Tamamlama (Tab)

Tab tuşu ile AI destekli kod tamamlama. Cursor, yazdığınız kodun context'ini anlayarak en uygun kod parçacığını önerir.

2

💬 Chat ile Kod Yazma (Cmd+K)

Doğal dille kod yazın. "React component oluştur" deyin, AI sizin için component'i yazacak.

3

🔍 Codebase Chat (Cmd+L)

Tüm projenizi analiz ederek proje-spesifik sorulara cevap verir ve kod önerilerinde bulunur.

4

🎨 Multi-file Editing

Birden fazla dosyayı aynı anda düzenleyerek tutarlı değişiklikler yapın.

⚙️ Kurulum ve İlk Ayarlar

📥 Kurulum Adımları

1

cursor.sh adresinden indirin

macOS, Windows ve Linux için mevcut

2

VS Code ayarlarınızı import edin

Cmd+Shift+P → "Import VS Code Settings"

3

AI modelini seçin

GPT-4, Claude Sonnet, ya da kendi API anahtarınız

# Terminal'den Cursor'ı başlatma
cursor .

# Veya proje klasörünü açma
cursor /path/to/your/project

🔄 Geliştirme İş Akışı

1

📋 Planlama

2

🏗️ Temel Yapı

3

💻 Kod Yazma

Tab ile hızlı kod tamamlama, Cmd+K ile düzenleme yapın.

4

🐛 Debug & Test

"Bu kodu test et" ve "Hataları bul" komutlarıyla kalite kontrolü yapın.

5

🚀 Deploy

Deploy scriptleri ve CI/CD pipeline'ları oluşturmak için AI yardımı alın.

💡 İpuçları ve Püf Noktaları

🎯

Spesifik Ol

📁

Context Sağla

Iterative Yaklaşım

Öğrenme Aracı

⌨️ Keyboard Shortcuts

Tab          → AI kod tamamlama
Cmd+K        → Inline chat (seçili kod için)
Cmd+L        → Codebase chat (proje geneli)
Cmd+Shift+L  → Chat geçmişi
Cmd+I        → AI panel toggle
